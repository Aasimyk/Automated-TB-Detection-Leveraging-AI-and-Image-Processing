About the Research

This project proposes an automated approach to detect pulmonary tuberculosis (TB) using chest X-rays (CXR). Under the mentorship of Dr. Ramesh Poonia, this work applies image processing techniques to improve diagnostic accuracy.

Methodology

  • Preprocessing: Wiener filtering and histogram equalization
  • Image Registration: Rigid alignment using histogram-based similarity
  • Segmentation: Watershed segmentation of lung regions

Datasets Used

  • JSRT: 247 chest X-rays (138 nodule, 93 non-nodule)
  • Montgomery County: 138 X-rays (80 normal, 58 TB cases)

Summary of Results

The proposed watershed segmentation method demonstrated improved sensitivity and specificity over traditional techniques like global thresholding and active contour.
